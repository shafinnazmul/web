01 Exam Strategy Summary
▶

The 50-minute midterm (Quiz 1, Mar 2) covered Lectures 1–15 with: formal proofs by induction/construction/contradiction, true/false asymptotic claims with formal proofs or counterexamples, function growth-rate ordering, pseudocode + complexity analysis, and one Dijkstra trace. The 120-minute final (Quiz 2, Apr 30) is comprehensive over all 27 lectures — same question types reappear plus new material on Master Theorem, Bellman-Ford, Floyd-Warshall, and NP-completeness reductions.

Study Prioritization Buckets ▶
Bucket Content Source
A — Midterm re-tested (~40%) Proof techniques, Big-O/Ω/Θ formal claims, growth-rate ordering, BFS vs DFS, Dijkstra trace, pseudocode patterns Quiz 1 patterns + A4 (graded)
B — New post-midterm (~45%) Master Theorem (Cases 1–3, non-applicable), Karatsuba/Strassen, Bellman-Ford, Floyd-Warshall, DAG topological sort + longest path, NP-completeness reductions A6 (graded) + Lec 16–27
C — Integration / challenge (~15%) Master Theorem + D&C runtime; NP reduction correctness + formal proof structure; graph algorithm complexity + adjacency representation choice Cross-topic themes (§05)
Question Format Distribution ▶
  • Formal proofs — induction, construction, contradiction (one of each likely)
  • True/False + counterexample for asymptotic claims
  • Function ordering (growth rates via limit-comparison method)
  • Algorithm trace — step-by-step table states (Dijkstra, BFS, Bellman-Ford)
  • Pseudocode + English explanation + complexity analysis (20 pts each in A4, A6)
  • Short proofs — graph properties, MST, NP reduction correctness (both directions)
  • Multiple choice with justification — Master Theorem case ID, graph complexity
